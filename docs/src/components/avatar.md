## API

### 属性

| 属性名 | HTML 属性 | 说明       | 类型     | 默认值   |
| ------ | --------- | ---------- | -------- | -------- |
| `name` | `name`    |            | `string` | `''`     |
| `size` | `size`    | 头像的尺寸 | `string` | `'2rem'` |
| `src`  | `src`     | 头像的地址 | `string` | `''`     |
